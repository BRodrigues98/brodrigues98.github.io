Project 2: Expert System for Cancer Study

An expert system to determine if a person is eligble to participate in the study

This was a group assignment where we had to implement an Expert System to determine if a patient with esophageal cancer was eligible to participate in a study based on several inclusion and exclusion criteria. This was implemented using Java and Drools.
